Exercise:

Write down what the following statements will return as a boolean. Try to figure this out before putting the commands in the browser console.

  • 2 == "2"
  • 2 === "2"
  • 2 === 2
  • "hello world"
  • ""
  • true
  • 0
  • -50
  • 50
  • 5 > 1
  • "1 > 10"
  • "false"

To check your answers, you can run the statements with the function `Boolean()`.
